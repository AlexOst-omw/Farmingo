# Farmingo

World of Warcraft Addon.
A lightweight and minimalistic farming tracker that records your loot, gold, and drop history across mobs, dungeons, and zones.
Farmingo is designed specifically for mob farming.
If you enjoy farming transmogs or rare drops, Farmingo helps make it more engaging by tracking how many kills it takes to get that item, along with your loot and gold progress.

FEATURES
Loot Tracking. Track how many times each mob has been looted and what items they drop. Monitor total and session gold earned from your farming runs per mob/zone/continent.
Location-Based Stats. See where your loot comes from - organized by zones, continents, dungeons, and raids.
Profiles System. Create multiple profiles and switch between them per character.
Search Function. Quickly find mobs or items using built-in search.
Tooltip Integration. Shows how many loots it took before an item dropped.
Clean & Customizable UI. Movable, resizable window with simple settings.

NOTES
Dungeon and raid mobs may appear as "Unknown" due to Blizzard API restrictions.
Subzones are normalized for better readability.

COMMANDS
/ft show           - Show window  
/ft hide           - Hide window  
/ft reset          - Request full data reset  
/ft confirmreset   - Confirm reset  
/ft profile        - Show active profile  
/ft profiles       - List profiles
